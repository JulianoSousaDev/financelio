🔄 Data Flow

1️⃣

User opens Dashboard

useMonthlySummary(month, year) → supabase.rpc('monthly_summary') → PostgreSQL
↓
2️⃣

Supabase processes RPC

monthly_summary(month, year, user_id) → SELECT SUM(amount) FROM transactions
↓
3️⃣

DashboardCard renders results

{ total_income, total_expense, net_balance } → Cards (green/red)
4️⃣

User adds transaction

Modal → Input validation → supabase.from('transactions').insert()
↓
5️⃣

RLS check then INSERT

RLS: auth.uid() = user_id → INSERT INTO transactions → fetchTransactions() refresh
